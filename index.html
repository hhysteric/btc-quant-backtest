MA / EMA 计算方式与择时逻辑

简单移动平均 SMA(n)：最近 n 根收盘价的算术平均。

SMA_t(n) = (C_t + C_{t-1} + … + C_{t-n+1}) / n

指数移动平均 EMA(n)：对近期价格加权更高，平滑系数 k = 2 / (n + 1)。首个值用前 n 根 SMA 作种子，其后递推：

k = 2 / (n + 1)
EMA_t = C_t × k + EMA_{t-1} × (1 − k)

择时信号：

  • 单均线：收盘价 > 均线 → 全仓持币；收盘价 < 均线 → 全部卖出空仓。
  • 双均线：短周期均线 > 长周期均线（金叉）→ 持币；短 < 长（死叉）→ 空仓。

C 为收盘价，下标 t 表示第 t 根 K 线。择时类策略用一次性初始资金满仓进出，不追加本金。
